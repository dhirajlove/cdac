Q.1 
int main() 
{ 
    int x = 2, y = 5; 
    (x & y) ? printf("True ") : printf("False "); // False
    (x && y) ? printf("True ") : printf("False "); // True
    return 0; 
}

A) False True
B) True False
C) True True
D) False False
Answer: A

2 -- 0000 0010
5 -- 0000 0101
---------------
     0000 0000 --> 0 -- False condition

2 && 5 ---> T && T ---> True condition
